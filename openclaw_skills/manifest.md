# The Poltergeist Skill Tree
*Productivity is forbidden. Magic is mandatory.*

This directory contains the "Sovereign Skills" for OpenClaw. 
To install a skill, copy the `.js` or `.ts` file into your OpenClaw `skills/` directory (or use the OpenClaw "Add Skill" feature if available).

## Skill Manifest

| Skill Name | Function | Trigger |

Ensure your `grok_relay.py` bridge is running so these skills can query Sophia for "Permission" or "Vibe Checks".
